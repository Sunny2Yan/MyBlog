## 变量与可变性
1. 变量(variables)：
    - 不可变变量：Rust 变量默认具有不可变性。变量一旦值被绑定一个名称上，就不能改变这个值。如 `let x = 5`；
    - 可变变量：在变量名前添加 mut 来使其可变，后续可对其值进行修改。如 `let mut y = 5`。
    - 未使用的变量需要以**下划线开头**，如 `let _z = 5`
2. 常量(constants)：
    - 不允许对常量使用 mut。
    - 声明常量使用 const 关键字而不是 let，并且必须注明值的类型。
    - Rust 对常量的命名约定是在单词之间使用全大写加下划线。
3. 隐藏(Shadowing)：即允许声明相同的变量名。
    - 定义一个与之前变量同名的新变量，则称第一个变量被第二个隐藏。

::: warning 隐藏与mut的区别：
隐藏会生成完全不同的新变量，两个变量只是名称相同，其中涉及一次内存对象的再分配；而 mut 声明的变量，可以修改同一个内存地址上的值，并不会发生内存对象的再分配，性能要更好。
:::

```rust
fn var(){
    let mut x = 5;  // 变量
    const THREE_HOURS_IN_SECONDS: u32 = 60 * 60 * 3;  // 常量

    let x = x + 1;  // 这个变量x隐藏了上面的x，此时不需要mut（mut是可改变变量值，而隐藏是创建新的变量）

    // 隐藏与mut的区别：
    let spaces_1 = "   ";
    let spaces_1 = spaces_1.len();  // 4。隐藏定义了一个新变量，涉及了内存对象分配

    let mut spaces_2 = "   ";
    // let spaces_2 = spaces_2.len();  // 报错。mut直接在原内存上修改，无法更改数据类型
}
```

## 变量解构

解构式赋值：可以在赋值语句的左式中使用元组、切片和结构体模式。

```rust
struct Struct {
    e: i32
}

fn main() {
    // 1. 复杂变量定义
    let (a, mut b): (bool,bool) = (true, false);
    println!("a = {:?}, b = {:?}", a, b);

    // 2. 解构式赋值
    let (a, b, c, d, e);

    (a, b) = (1, 2);  // 赋值左式中使用元组，let自动匹配
    [c, .., d, _] = [1, 2, 3, 4, 5];  // 赋值左式中使用切片，_ 代表匹配一个值（同python）
    Struct { e, .. } = Struct { e: 5 };  // 赋值左式中使用结构体

    assert_eq!([1, 2, 1, 4, 5], [a, b, c, d, e]);
}
```
